# CLIで動くtodoアプリ

UIとか考えるのめんどいのでこれで

## config.jsonの使い方
```json
{
    "language": "en"
}
```
''en'の部分を'ja'に変更すると日本語にできる

## インストールと使用方法の確認

1. GitHubリポジトリをクローンします。

```sh
git clone https://github.com/riku-yanagihashi/CLI-todo-app.git
cd CLI-todo-app
```

2. パッケージをインストールします
```
pip install .
```

3. 'todo'コマンドで実行
```
todo
```
